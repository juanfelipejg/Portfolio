Ferro

.Net Core y Xamarin
Puedes verlo en: https://github.com/jjaramga/Ferro

Ferro es una aplicación movíl desarrollada para facilitar la gestion de turnos y solicitudes de los conductores del Metro de Medellín. La aplicación se desarolló en Xamarin Forms con el fin de aprovechar la aplicación en las plataformas mas usadas hoy en dia para dispositivos (Android- iOS). Así mismo se desarrolló un sitio web en .Net Core para apoyar los procesos administrativos que conlleva este servicio.

Los proyectos de desarrollo de software están siempre susceptibles a cambios generados por las necesidades de nuestro cliente o la evolución del mercado por lo cual es importante adecuarse a las nuevas exigencias y que mejor que con Scrum. Scrum es un marco de trabajo que implementa buenas practicas con el fin de destacar el trabajo en equipo y obtener excelentes resultados asociados a los tiempos de entrega, costes y calidad.

Git y GitHub

Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. En su lugar GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones Git. GitHub sería la red social de código para los programadores, tu propio curriculum vitae.
